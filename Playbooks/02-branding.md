# Playbook 02 — Branding & Identidade Visual

> **Pulso Onboarding Framework** | Etapa 2 de 12

---

## Objetivo

Criar ou reformular a identidade visual da clínica/consultório para transmitir credibilidade, confiança e diferenciação. A marca é a base de toda comunicação posterior.

---

## Quando Executar

- Após conclusão do Diagnóstico (Playbook 01)
- Duração estimada: 5–7 dias
- Entregável: Manual de Identidade Visual (MIV)

---

## Brief de Branding

### Perguntas-chave para o cliente

1. Qual é a personalidade da sua clínica? (3 adjetivos)
2. Como você quer que os pacientes se sintam ao ver sua marca?
3. Tem alguma referência visual que admira? (clínicas, hotéis, apps de saúde)
4. Cores que você NÃO quer usar?
5. A marca deve transmitir: ( ) Tecnologia ( ) Humanidade ( ) Tradição ( ) Inovação
6. Público-alvo principal tem perfil: ( ) Popular ( ) Classe Média ( ) Premium

## Entregáveis do MIV

### Pacote Básico (obrigatório)
- [ ] Logotipo principal (PNG + SVG)
- [ ] Versão horizontal
- [ ] Versão compacta/ícone
- [ ] Versão monocromática (preto)
- [ ] Versão monocromática (branco)
- [ ] Guia de cores (HEX + RGB + CMYK)
- [ ] Guia de tipografia
- [ ] Área de proteção (clear space)
- [ ] Usos incorretos (5 exemplos do que NÃO fazer)

### Pacote Completo (recomendado)
- [ ] Papelaria: cartão de visita, papel timbrado
- [ ] Assinatura de e-mail
- [ ] Template stories Instagram
- [ ] Template post feed Instagram
- [ ] Favicon
- [ ] Mockups de aplicação (uniforme, placa, fachada)

---

## Processo de Criação

```
1. Brief do cliente                (30min call)
2. Pesquisa de referências         (interno, 2h)
3. Rascunhos conceituais           (3-5 direções)
4. Apresentação de 2 direções      (call com cliente)
5. Refinamento da direção escolhida (2-3 iterações)
6. Entrega do MIV completo
7. Arquivos finais (todos os formatos)
```

---

## Critérios de Qualidade

- [ ] Legível em tamanhos pequenos (16px mínimo)
- [ ] Funciona em fundo claro E escuro
- [ ] Contraste WCAG AA (mínimo 4.5:1 para texto)
- [ ] Coerência com o posicionamento definido no diagnóstico
- [ ] Diferenciação dos top 3 concorrentes mapeados
