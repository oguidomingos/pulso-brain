# Playbook 11 — Análise de Conta (Setup de Agentes)

> **Pulso Onboarding Framework** | Etapa 11 de 12

---

## Objetivo

Realizar uma análise profunda da conta do cliente (marketing, negócio, concorrência, narrativa de marca) para alimentar os agentes especializados da Pulso com contexto rico, garantindo execuções cada vez mais precisas e alinhadas.

---

## Quando Executar

- Ao início de cada novo cliente (primeiros 3 dias)
- Na renovação anual de contrato
- Quando houver mudança significativa no negócio do cliente
- Antes de escalar budget de tráfego pago

---

## Framework de Análise (5 Dimensões)

### Dimensão 1 — Negócio

```
1.1 Modelo de negócio
    → Tipo: Clínica própria / Franquia / Consultório PJ / Hospital
    → Faturamento estimado (faixa): até R$50k / R$50–200k / R$200k+/mês
    → Sazonalidade: existe pico em algum mês?
    → Planos de saúde atendidos: [lista]
    → Procedimentos mais lucrativos: [ranking]

1.2 Estrutura da equipe
    → Número de médicos/profissionais
    → Tem recepcionista/secretária?
    → Tem coordenador de marketing interno?
    → Tem gestor de tráfego próprio?

1.3 Tecnologia em uso
    → Sistema de agendamento: [Software]
    → CRM: [Software ou nenhum]
    → Prontuário eletrônico: [Software]
    → WhatsApp Business: SIM / NÃO
```

### Dimensão 2 — Marketing Atual

```
2.1 Histórico de campanhas
    → Já rodou Meta Ads? [SIM/NÃO] → Resultados: [...]
    → Já rodou Google Ads? [SIM/NÃO] → Resultados: [...]
    → Maior aprendizado: [...]
    → Maior fracasso: [...]

2.2 Presença digital atual
    → Instagram: @[handle] | X seguidores | Y% engajamento
    → Google Meu Negócio: X estrelas | Y reviews
    → Site: [URL] | PageSpeed mobile: X | PageSpeed desktop: X
    → YouTube: [existe? URL]
    → TikTok: [existe? URL]

2.3 Orçamento disponível
    → Budget total/mês para marketing: R$ [X]
    → Distribuição ideal: Meta X% / Google X% / Produção X%
```

### Dimensão 3 — Público e ICP

```
3.1 Perfil do Paciente Ideal (ICP)
    → Faixa etária: [X–Y anos]
    → Gênero predominante: [M/F/ambos]
    → Renda familiar: [faixa]
    → Localização: [bairros/cidades]
    → Plano de saúde: [quais]

3.2 Jornada do Paciente
    → Como descobre a clínica? (busca, indicação, social)
    → Tempo entre descoberta e agendamento: [dias]
    → Principal objeção antes de agendar: [...]
    → Por que cancela? [...]
    → Por que retorna? [...]

3.3 Personas documentadas
    → Persona 1: [Nome, idade, perfil, dor, objetivo]
    → Persona 2: [Nome, idade, perfil, dor, objetivo]
```

### Dimensão 4 — Competição

```
4.1 Mapeamento de concorrentes diretos
    | Concorrente | Site | Instagram | GMB | Pontos fortes | Pontos fracos |
    |-------------|------|-----------|-----|---------------|---------------|
    | 1. | | | | | |
    | 2. | | | | | |
    | 3. | | | | | |

4.2 Análise de anúncios da concorrência
    → Meta Ads Library: [captura das peças ativas]
    → Google: [busca pelas palavras-chave principais]
    → Diferenciais que nenhum concorrente comunica: [...]

4.3 Oportunidade de posicionamento
    → Nicho inexplorado: [...]
    → Especialidade sem concorrência local: [...]
    → Canal de aquisição sem concorrência: [...]
```

### Dimensão 5 — Narrativa de Marca

```
5.1 Essência da marca
    → Missão (por que existem?): [...]
    → Visão (onde querem chegar?): [...]
    → Valores (3–5 palavras): [...]
    → Tom de voz: Formal/Informal | Técnico/Acessível | Frio/Quente

5.2 Arquétipo da marca
    → Identificar o arquétipo dominante:
      [ ] Herói (vence o problema)
      [ ] Cuidador (cuida do paciente)
      [ ] Sábio (compartilha conhecimento)
      [ ] Criador (inovação em saúde)
      [ ] Governante (autoridade máxima)

5.3 Proposta de valor única (UVP)
    → Completar: "Para [ICP] que [dor/necessidade],
      [Nome da clínica] é o único [categoria] que [benefício único]
      porque [razão de acreditar]."
```

## Output da Análise

### Documento de Briefing de Conta
- Resumo executivo (1 página)
- As 5 dimensões preenchidas
- Top 3 prioridades estratégicas
- Quick wins (pode fazer esta semana)
- Riscos identificados

### Alimentação dos Agentes
Este briefing deve ser carregado no contexto de cada agente Pulso:
- Web Builder: recebe Dimensões 1, 3, 5
- Content Creator: recebe Dimensões 3, 5
- Ads Manager: recebe Dimensões 2, 3, 4
- Sales Strategist: recebe Dimensões 1, 2, 9 (proposta)

## Cases Reais: Análise de Conta nos POCs

### Case 1 — Clínica Modelo Ortopédica

**Análise executada (Dimensão 1 — Negócio):**
- Tipo: Clínica própria (consultório PJ)
- Faturamento estimado: R$50–100k/mês
- Sazonalidade: sem pico definido
- Procedimento mais lucrativo: infiltração articular (maior ticket, alta demanda)
- Estrutura: 1 médico + 1 recepcionista + nenhuma área de marketing interna

**Análise executada (Dimensão 3 — ICP):**
- Faixa etária: 45–70 anos (predominância feminina, 60%)
- Como descobre: Google Maps (principal) + indicação médica
- Principal objeção: "Já fiz tratamento antes e não funcionou"
- Por que retorna: resultado rápido (infiltração tem retorno visível em 48h)

**Persona documentada:**
> **"Dona Maria, 58 anos"** — sofre de artrose no joelho há 3 anos. Ativa no WhatsApp, usa Google para buscar médicos. Tem plano de saúde mas prefere particular quando tem urgência. Maior medo: cirurgia.

**UVP definida:**
> "Para pacientes acima de 45 anos com dor no joelho ou quadril, somos o único consultório de ortopedia em [cidade] que oferece infiltração articular guiada com resultado em 48h, sem cirurgia, com atendimento no mesmo dia."

**Arquétipo:** Cuidador + Sábio (confiança + conhecimento)

### Case 2 — Dr. Leandro Marques Dutra

**Análise executada (Dimensão 2 — Marketing):**
- Histórico Meta Ads: Nunca rodou (Pixel instalado mas sem eventos)
- Histórico Google Ads: Nunca rodou
- Instagram: 3 posts/mês, sem estratégia
- Site PageSpeed mobile: 42/100 (crítico)

**Análise executada (Dimensão 4 — Competição):**
- 3 concorrentes mapeados na Meta Ads Library
- Todos comunicavam "tratamento de dor" de forma genérica
- Nenhum usava ângulo "sem cirurgia" como diferencial no copy
- Oportunidade identificada: posicionamento como alternativa à cirurgia

**UVP resultante:**
> "Para pacientes entre 40–65 anos sofrendo de dor crônica há mais de 3 meses, Dr. Leandro oferece diagnóstico especializado em 1 consulta com protocolo multidisciplinar — sem indicar cirurgia quando não é necessário."

**Arquétipo:** Sábio + Herói (o médico que resolve onde outros falharam)

**Insight crítico da análise:** O médico atendia 2 especialidades mas comunicava apenas 1 (ortopedia). A análise revelou que dor crônica era mais lucrativa (sessões recorrentes) e menos competida no digital. Decisão: criar campanha separada para dor crônica.

---

### Case 3 — Santa Isabel (Reabilitação)

**Análise executada (Dimensão 5 — Narrativa):**
- Missão: "Devolver movimento e independência para quem acreditou que não conseguiria mais"
- Valores: Cuidado, Experiência, Esperança, Multidisciplinaridade
- Tom de voz: Acolhedor + Profissional (não frio, não informal)
- História única: 46 anos de história, 2.000+ pacientes, maior centro DF

**UVP resultante:**
> "Para pacientes pós-cirúrgicos em Brasília que precisam de reabilitação de qualidade, Santa Isabel é o único centro multidisciplinar com 46 anos de experiência que oferece recuperação completa com equipe de fisio, TO e fonoaudiologia sob o mesmo teto."

**Arquétipo:** Cuidador + Governante (cuidado com autoridade máxima)

**Diferencial competitivo descoberto na análise:**
- Nenhum concorrente em Brasília comunicava "multidisciplinar sob mesmo teto"
- "46 anos" nunca foi usado em comunicação antes (3 logos desatualizados em uso)
- 2.000+ pacientes era número desconhecido até o levantamento interno
