## Notas de Uso

1. **Estes benchmarks são baseados em 3 POCs** (amostra pequena). Atualizar a cada 2 novos clientes com dados reais.
2. **CPCs variam por cidade:** Capitais (SP, RJ, BSB) tendem a ter CPC 20–40% maior.
3. **Sazonalidade:** Jan–Mar é período de maior demanda para fisioterapia (início de ano, novas metas). Jul–Ago para ortopedia (inverno, dores articulares).
4. **Benchmarks de ROAS** assumem ticket médio de R$250–500/consulta. Ajustar para especialidades com ticket diferente (cirurgia plástica: R$5.000+, dermatologia estética: R$800+).
